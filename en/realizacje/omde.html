Retail Installation Planning for Media Markt · Saturn

Installation app
for Media Markt and Saturn

A system for Media Markt and Saturn employees and OMDE installers. Sales staff can immediately see when and where a device can be installed.

0 s
salesperson’s wait time for available installation slots
2 roles
in one system: store salespeople and OMDE installers
The entire city
real-time view of installation zones and available time slots

Challenge

OMDE sp. z o.o. provides installation services for the Media Markt and Saturn chains. When selling a device requiring installation (home appliances, air conditioning, satellite systems), a store salesperson had to manually contact the OMDE dispatcher to schedule an appointment. This process was slow, error-prone, and frustrating for both the salesperson and the customer waiting at the checkout.

The need was simple: the salesperson should be able to see immediately when an installer could arrive in a given city district.

Solution

We built a real-time web application accessible via a browser—no installation required, on any device:

  • Salesperson view — the salesperson enters the customer’s address, and the system immediately displays the nearest available installation slots in that city zone. Booking takes just a few seconds.
  • Zone management — the city is divided into installation zones. Each zone has assigned installers, schedules, and daily capacity.
  • OMDE Dispatcher Panel — order management, assigning installers, viewing availability on a map and calendar.
  • Installer app — list of orders for the day, navigation to the customer, confirmation of installation completion with a signature.

“Previously, scheduling an appointment took several minutes and required a phone call. Now the salesperson does it themselves in 20 seconds, and the customer leaves the store with a confirmed appointment.”

How we did it

The project required a deep understanding of OMDE’s logistics operations—how routes are planned, how zones work, and what capacity constraints exist. For two weeks, we worked closely with dispatchers to understand the actual workflow.

Key design decision: the app had to work instantly even with a weak internet connection in the store. We optimized every database query and implemented schedule caching to ensure response times didn’t exceed 200 ms.

Results

  • Time to book an installation appointment reduced from several minutes to 20 seconds
  • Elimination of booking errors resulting from phone communication
  • Salespeople no longer have to wait for a dispatcher—they see availability immediately
  • The OMDE dispatcher sees the full schedule of installers in a single view
  • The system supports all Media Markt and Saturn stores within OMDE’s coverage area
