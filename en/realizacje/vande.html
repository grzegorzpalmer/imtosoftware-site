The Challenge

Procurement and tendering departments in manufacturing companies process dozens of offers from different suppliers every day. Each one arrives in a different format — PDF, scan, Excel spreadsheet, Word document — with a different structure, varying parameter names, and different table layouts. Manual comparison takes hours, and errors caused by fatigue or differences in interpreting terms cost companies real money.

The problem runs deeper: there is no standard for commercial and tender offers. Every supplier sends something different, so off-the-shelf data extraction tools fail at the first non-standard table. Companies need a system that understands context — not just scans columns.

The Solution

We built Vande — an AI-powered SaaS platform that automates the entire process of analyzing and comparing offers — from document upload to purchasing decision:

Analysis of documents with unknown structure

The AI recognizes the document type, categorizes it, and extracts key fields even from documents with a completely new layout — without the need to define templates.

Simultaneous comparison of multiple offers

The system compares multiple offers at once, taking into account all parameters and conditions defined by the user. Similarities and advantages of each offer are clearly displayed.

Weighted parameter scoring

Each comparison parameter (price, delivery time, warranty conditions, etc.) can have a different weight. The system automatically calculates a final score for each offer.

Negotiation support and notes

The platform enables conducting negotiations directly within the system — sending questions to suppliers, creating notes, and tracking communication history for each offer.

One-off request or full project

Each comparison can be a quick, one-off request to the system or carried out as part of a broader project with full decision-making process support and version history.

High recognition confidence

Average field recognition confidence is 94%, and each field has an assigned confidence level with warnings for values requiring verification. Processing time: ~5 seconds.

"Offers from different suppliers, different formats, different naming — the system reads them all and delivers a ready-made comparison in seconds. What used to take hours now happens automatically."

Stack and Architecture

Vande runs as a SaaS web application deployed on Microsoft Azure (Azure Web Apps). The architecture is based on an AI pipeline for document extraction and classification, a comparison module with configurable weighted parameters, and a layer for project management and supplier communication. Supported input formats: PDF, JPG, PNG, DOCX, TIFF, email.

Project Status

The project is currently at the pilot stage — the live environment confirms the key assumptions: automatic recognition of document types (invoices, reports, OHS instructions, attendance cards), field extraction with high confidence, and categorization. Work is underway on the offer comparison module, weighted scoring, and negotiation features.

IMTO Software is responsible for the entire scope: system architecture, development, AI model integration, and deployment on the Azure platform.
